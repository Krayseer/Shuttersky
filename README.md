# ShutterSky

## Описание
Приложение для облачного хранилища фотографий. Позволяет загружать фотографии и получать их по сгенерированному после загрузки уникальному идентификатору.

## Методы для работы с приложением
- `GET ("/photo/{uuid}")` получить фотографию по идентификатору uuid.
- `POST ("/photo")` загрузить фотографию. В параметр "photo" нужно добавить необходимую для загрузки фотографию. Возвращается уникальный идентификатор, по которому можно будет получить фотографию.
